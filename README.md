Install original version
------------------------

    $ virtualenv venv
    $ . venv/bin/activate
    $ pip install -e hg+https://bitbucket.org/mirror/suds@1e48fd79a1fc#egg=suds

Install hacked version
----------------------

    $ virtualenv venv
    $ . venv/bin/activate
    $ pip install -e hg+https://bitbucket.org/saltycrane/suds#egg=suds
